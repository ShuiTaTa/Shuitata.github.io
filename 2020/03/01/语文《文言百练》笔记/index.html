1.赵人患鼠

赵人患鼠，乞$^{求，讨}$猫于中山。中山人予之。猫，善捕鼠及鸡。月余，鼠尽，而其鸡亦尽。其子患之，告其父曰：“盍$^{何不}$去$^{驱赶，使离开}$诸$^{之乎，它呢}$？”其父曰：“是$^{这}$非$^{不是}$若所知也。吾之患在鼠，不在乎无鸡。夫有鼠，则窃吾食，毁吾衣，穿吾垣墉$^{yu\acute{a}n\ y\bar{o}ng}$$^{墙壁}$ ，毁伤吾器用，吾将饥寒焉，不病$^{损害}$于$^{比}$无鸡乎？$^{不于无鸡病乎？不是比没有鸡损害更大吗？}$无鸡者，弗食鸡则已耳，去$^{距离}$饥寒犹远，若之何$^{为什么}$而去$^{驱赶，使离开}$夫$^{那}$猫也！”

患

动词：

  • 担忧，担心：
    • 赵人患鼠。
    • 其子患之。
    • 燕太子丹患秦兵至国。
    • 患得患失。
  • 得病：
    • 时有患疟疾者。
    • 患病，患者。

名词：

  • 忧患、灾祸：
    • 吾之患在鼠，不在乎无鸡。
    • 国日久，内外无患。
    • 养虎遗患。
  • 疾病：
    • 愈膏肓之患难。
  • 毛病、弊病：
    • 人之患在好为人师。

若

动词：

  • 像，如，好像：
    • 若寡人者，可以保民哉？
    • 上善若水。
    • 安之若素。
    • 若无其事。

代词：

  • 人称代词——你，你的：
    • 是非若所知也。
    • 更若役，复若赋，则何如。
  • 指示代词——这个，此，如此，这些：
    • 以若所为，求若所欲，犹缘木而求鱼也。

连词：

  • 假如，如果：
    • 天若有情天亦老。
  • 或；和：
    • 时有军役若水旱，民不困乏。
    • 愿取吴王若将军头，以报父仇。
  • 若之何：为什么，怎么样。
    • 若之何而去夫猫也！
  • 若夫：句首语气，引起下文，“至于说到” 的意思。
    • 若夫**霪雨霏霏**，连月不开。

诸

  • 众多，各：
    • 诸位。
    • 诸将。
    • 诸子百家。
  • 第三人称代词——之：
    • 晋荐饥，使乞求籴于秦。秦伯谓子桑：“与诸乎？”
  • 兼词——之乎：
    • 盍去诸？
    • 子闻诸？

成语

  • 患得患失：在没得到以前怕得不到，得到以后又怕失去。形容人的得失心很重。
  • 养虎遗患：饲养老虎，最后为自己带来灾祸。比喻不除去仇敌、恶人，将给自己留下后患。
  • 好为人师：喜欢做别人的老师。指人不谦虚，喜欢教导别人。
  • 安之若素：在反常或危困等情况下，仍安然处之，毫不在意。
  • 缘木求鱼：爬到树上去抓鱼。比喻用错方法，徒劳无功。
  • 霪雨霏霏：连绵不断的雨。

2.周处改过自新

周处年少时，凶强侠气$^{任性使气。这里有“好争斗”的意思}$，为乡里所患。又义兴水中有蛟，山中有白额虎，并皆暴犯$^{侵害}$百姓。义兴人谓为三横$^{三害。横:横暴，放纵。}$，而处尤剧$^{厉害，严重}$。或说$^{劝说}$处杀虎斩蛟，实冀$^{希望}$三横唯余其一。处即刺杀虎，又入水击蛟。蛟或浮或没，行数十里，处与之俱。经三日三夜，乡里皆谓已死，更相$^{交互，互相}$庆。竟杀蛟而出，闻里人相庆，始知为人情${人心，民心}$所患，有自改意。乃入吴寻二陆。平原不在，正见清河，具以情告，并云自修改而年已蹉跎$^{荒废，光阴虚度}$，终无所成。清河曰：“古人贵$^{意动，以…为贵}$朝闻夕死，况君前途尚可。且人患志之不立$^{之，主谓之间，取独}$，何忧令$^{美好的。巧言令色}$名不彰邪？”处遂改励，终为忠臣孝子。

或

动词：

  • 通“惑”，迷惑，不理解：
    • 或失道。
    • 臣甚或之。

代词：

  • 有的（代人或事物）：
    • 或说处杀虎斩蛟，实冀三横唯余其一。
    • 左右或欲刃相如。
    • 及昭宗时，尽杀朝之名士，或投之黄河。
    • 回视日观以西峰，或得日，或否。
    • 或师焉，或不焉，小学而大遗，吾未见其明也。
    • 兵刃既接，弃甲曳兵而走。或百步而后止，或五十步而后止。以五十步笑百步，则何如？

连词：

  • 或者：
    • 君子之道，或出或处，或默或语，二人同心，其利断金。
    • 或多或少。
  • 如果、假设：
    • 所守或匪亲，化为狼与豺。

副词：

  • 有时：
    • 其神或岁不至，或岁数来。
    • 或命巾车，或棹孤舟。
  • 或许，也许（不确定性）：
    • 云霞明灭或可睹。

3.晁错削藩

错所更$^{改变，更改}$令三十章，诸侯皆喧哗，疾$^{憎恨，怨恨}$晁错。错父闻之，从颍川来，谓错曰：“上初即位，公为政$^{从政，治理国家}$用事$^{执掌政权}$，侵削$^{侵夺}$诸侯，别$^{使动}$疏$^{使动}$人骨肉，人口议多怨公者，何也？”晁错曰：“固$^{本来}$也。不如此，天子不尊，宗庙不安。”错父曰：“刘氏安矣，而晁氏危矣，吾去$^{离开}$公归$^{走了，回去了}$矣！”遂饮药死，曰：“吾不忍见祸及吾身。”死十余日，吴楚七国果反，以诛错为名。及窦婴、袁盎进说$^{臣子向君王陈说}$，上令晁错衣$^{动词}$朝衣斩东市。

更

动词$g\bar{e}ng$ :

  • 改变，更换：
    • 君子之过也，如日月之食焉：过也，人皆见之；更也，人皆仰之。
    • 自力更生。
    • 万象更新。
    • 改弦更张。
  • 交替，轮换：
    • 譬犹水火相更也，水盛胜火，火盛胜水。
  • 经历：
    • 人更三圣，世历三古。
    • 少不更事。

名词$g\bar{e}ng$ :

  • 古代夜间计时单位，一夜分为五更，每更约两小时：
    • 三更灯火五更鸡，正是男儿读书时.
    • 更漏。

副词$g\grave{e}ng$ :

  • 复，再：
    • 莫辞更坐弹一曲，为君翻作琵琶行。
  • 又，另：
    • 于是帝尧乃求人，更得舜。
  • 更加：
    • 白头搔更短，浑欲不胜簪。

固

形容词：

  • 坚固，稳固，牢固，（地势）险要牢固：
    • 本固枝荣。
    • 固若金汤。
    • 秦孝公据殽函之固，拥雍州之地。
  • 固执；顽固：
    • 汝心之固，固不可彻。
    • 固执己见。

动词：

  • 巩固，（形使动）使……固：
    • 晋始伯而欲固诸侯,故解有罪之地以分诸侯。
    • 固国不以山溪之险。
    • 固本清源。

副词：

  • 坚决，坚持：
    • 蔺相如固止之。
    • 固辞不受。
  • 本来，原来：
    • 生乎吾前，其闻道也固先乎吾。
    • 臣固知王之不忍也。
  • 通“故”，旧的：
    • 故步自封=固步自封.
